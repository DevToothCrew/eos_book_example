EOSJS 환경구성
1. nodeJS 설치

- sudo apt update
- sudo apt install node


2. npm 설치

- sudo apt install npm

3. EOSJS 설치

- npm install eosjs@16.0.9

4. create-react-app 설치

- sudo npm install -g create-react-app

5. PC Scatter 설치

- https://get-scatter.com

6. Scatter.JS 설치

- npm i -S scatterjs-core

- npm i -S scatterjs-plugin-eosjs
